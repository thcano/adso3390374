01-Emmet

Plugin for faster HTML and CSS coding

01-Child >

Creates child elements (nesting).

Example:

ul>li>a↔️

Result:

<ul> <li><a href=""></a></li> </ul>

02-Sibling +

Creates sibling elements.

Example:

header+section+footer↔️

Result:

<header></header> <section></section> <footer></footer>

03-Multiplication *

Repeats elements multiple times.

Example:

ul>li*3 ↔️

Result:

<ul> <li></li> <li></li> <li></li> </ul>

04-Class .

Selects all elements that have the specified class.

Examples:

div.contenedor↔️

Result:

<div class="contenedor"></div>

05-Classes ...

Creates multiple class attributes.

Examples:

div.clase1.clase2.clasen ↔️

Result:

<div class="clase1 clase2 clasen"><div>

06-ID Selector #

Selects a single element with the specified ID.

Example:

nav#menu ↔️

Result:

<nav id="menu"></nav>

07-Insert Text {}

Inserts the specified text content inside the element.

Example:

div{Lorem Ipsum Dolor} ↔️

Result:

<div>Lorem Ipsum Dolor</div>

08-Custom Attribute []

Creates custom attributes.

Example:

a[href=www.google.com]↔️

Result:

<a href="www.google.com"></a>

09-Item Numbering $

Adds consecutive numbers.

Example:

ul>li.num$*3↔️

Result:

<ul> <li class="num1"></li> <li class="num2"></li> <li class="num3"></li> </ul>

10-Reverse Numbering @-

Inverts numbering order.

Example:

ul>li.num$@-3↔️

Result:

<ul> <li class="num3"></li> <li class="num2"></li> <li class="num1"></li> </ul>

11-Base Numbering @n

Defines a base number for sequences.

Example:

ul>li.num$@3*4↔️

Result:

<ul> <li class="num3"></li> <li class="num4"></li> <li class="num5"></li> <li class="num6"></li> </ul>

12-Climb Up ^

Moves up one level.

Example:

section>article>div^p↔️

Result:

<section> <article> <div></div> </article> <p></p> </section>

13-Grouping ( )

Groups elements together.

Example:

(header>nav)(section>article)↔️

Result:

<header> <nav></nav> </header> <section> <article></article> </section>

14-Lorem lorem

Generates placeholder text.

Example:

lorem↔️

Result:

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Cum molestias in doloribus quidem impedit ut veniam rerum id veritatis, ullam sed ea fugit ipsam blanditiis. Provident animi quo atque deleniti.

15-HTML Documents html:

Creates various HTML document types.

Example:

html:xs↔️

Result:

<html:xs> </html:xs>

16-Meta Tags ( )

Generates meta tags.

Example:

meta:vp↔️

Result:

<meta name="viewport" content="width=device-width, initial-scale=1.0">
